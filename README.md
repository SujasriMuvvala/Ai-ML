# AIML Lab 1 - Feature Extraction

## Objective
To understand how to convert raw data (text and images) into numerical features for machine learning.

## Concepts Covered
- Text preprocessing
- N-grams (Unigram, Bigram)
- Feature extraction from images
- MNIST dataset analysis

## Observations
- Bigrams are better than unigrams for language detection
- Hole pixels effectively distinguish digits (0 vs 1)
- Combining features improves classification

## Tools Used
- Python
- NumPy
- Matplotlib
- NLTK
- Keras
